Class: exports

exports(conf)

new exports(conf)

Contructor method of the Config object

Parameters:
Name Type Description
conf Object

the list of configuation param you want to add.

  • BLOCK_MAX_SIZE : max size of the block, default is 2000;
  • BLOCK_MIN_DIFFICULTY : minimum difficulty for hashing, default is 3;
  • BLOCK_HASH_RATE_AVERAGE : number of blocks to use to determine the average of hash rate and calculate difficulty, default is 1;
  • BLOCK_MAX_DIFFICULTY : max difficulty, default is 9;
  • BLOCK_HASH_RATE_BY_DAY : numb of block we want to make by day ,default is 24;
  • BLOCK_HASH_METHOD : the hash method to use the chain, 'SHA265' or 'MD5', default is 'SHA256';
  • TX_CONTENT_EXPIRATION_HOURS : how many hours is store the content in a transaction before being cleaned? default is 24;
  • TX_CONTENT_MAX_SIZE_KO : max size in KO for a content, default is 250;
  • TX_MAX_SIZE_KO : max size in Ko of a transaction if we take off the content, default is 0.75;
  • BLOCK_MAX_SIZE_KO : max size in Ko for teh block, default is 300;
  • FOUNDERS : list of the public of blockchain founders, used for future administration of the chain.
  • GENESIS_BLOCK : the params of the genesis block of the chain, heigth 0
  • MONEY_BY_BLOCK : the money created by block, default is 15.
  • MONEY_BY_KO : the money by Ko to post a content, default is 1.
  • TX_FEE_MINE_MONEY : tax fee for mining a transaction of money, as ratio, default is 0.1.
  • TX_FEE_MINE_OWNERSHIP : tax fee for mining a transaction of ownership, as ratio, default is 0.1.
  • TX_FEE_MINE_CONTENT: tax fee for mining a transaction of content, as ratio, default is 0.25.
